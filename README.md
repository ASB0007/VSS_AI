# 🌌 Виртуальная Солнечная Система

Интерактивное веб-приложение с 3D-моделями планет и встроенным ИИ-чатом.

### 🚀 Возможности:
- Просмотр 3D моделей всех планет Солнечной системы
- Информация о каждой планете (расстояние, температура, диаметр и т.д.)
- Интерактивный чат с искусственным интеллектом (Gemini)
- Возможность скачать приложение для Windows, macOS, Linux, Android и iOS

### 🛠 Технологии:
- Python
- Gradio
- Flask
- Gemini API (Google Generative AI)
